# my.home.api

A RESTful API built for the my.home project.

## Description
This service provides a simple echo endpoint and is designed to be used as the backend for the my.home.web frontend.

## Configuration
- **Port:** The API runs on port 8081 by default. You can change this by setting the `PORT` environment variable.

## Usage
- Start the server: `npm start`
- Run tests: `npm test`
